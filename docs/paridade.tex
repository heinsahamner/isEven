\section*{Paridade}

Um número inteiro $n$ é dito \textbf{par} quando o resto da divisão por 2 é igual a zero.

\[
n \text{ é par} \iff n \bmod 2 = 0
\]

Equivalentemente, em representação binária, $n$ é par se o seu bit menos significativo for igual a 0.
